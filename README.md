# Obsidian Tasks Calendar Wrapper

This is a sample wrapper for Obsidian-Tasks-Calendar (https://github.com/702573N/Obsidian-Tasks-Calendar) and Obsidian-Tasks-Timeline (https://github.com/702573N/Obsidian-Tasks-Timeline).

This plugin provides setting pages and update support for [702573N](https://github.com/702573N)'s great work.
